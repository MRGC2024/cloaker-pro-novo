Link para Ads

Gerar novo link para Ads

Cole o link da sua landing page abaixo e clique em Gerar link. Use o link gerado como URL de destino nos seus anúncios (Meta Ads, Google Ads, etc.).

✅ Link gerado – use esse link nos seus anúncios:

            

            
Não precisa colocar nenhum script no seu site. Só use esse link como URL de destino nos Ads.

          

        

      

      

        
 Como funciona

        

          

            1. Em "Meus Sites", cadastre seu site e coloque a URL da sua landing page (o link real do seu site).

            2. O sistema gera um novo link para você (ex: https://seu-painel.up.railway.app/go/abc12xyz).

            3. Use esse novo link como URL de destino nos seus anúncios (Meta Ads, Google Ads, etc.).

            4. Quem clicar no anúncio passa primeiro pelo nosso servidor: se for permitido (ex: mobile, Brasil), vai para seu site; se for bloqueado (desktop, bot, etc.), vai para a URL de redirecionamento (ex: Google).
          

          

            
📌 Importante

            
    
              
  • Não precisa colocar nenhum script no seu site. Só usar o link gerado nos Ads.
    • 
              
  • Cada site tem seu próprio link e configurações (país, bloqueio de desktop, etc.).
    • 
              
  • Parâmetro de rastreamento: use o parâmetro ref=... no Meta Ads (Rastreamento → Parâmetros de URL) para que só quem clicar no anúncio seja permitido; quem acessar o link sem esse parâmetro será bloqueado.
    • 
              
  • Os parâmetros do anúncio (UTM, fbclid, etc.) são repassados para sua landing page.
    • 
              
  • Todos os acessos (permitidos e bloqueados) aparecem no painel.
    •

 Meu domínio do Cloaker

        

        

          
Configure a URL base dos seus links e o prefixo usado ao gerar novos links. Links já criados continuam funcionando em qualquer prefixo (/go/, /r/, /l/, /v/).

 Instruções de configuração DNS

        

          
    
            
  1. No Railway (obrigatório): Projeto → seu serviço → Networking → + Custom Domain → adicione o domínio (ex: cloaker.seudominio.com ou iniiciopropo.sbs). Sem isso, o domínio não chega ao seu app.
    2. 
            
  2. Tenha um domínio (ex: meusite.com.br). Você pode usar um subdomínio (ex: cloaker.meusite.com.br ou go.meusite.com.br).
    3. 
            
  3. Acesse o painel do seu provedor de DNS (onde o domínio está registrado: Registro.br, Cloudflare, GoDaddy, HostGator, etc.).
    4. 
            
  4. Crie um registro CNAME:
              
      
                
    • Nome/Host: o subdomínio que você quer usar (ex: cloaker ou go). Se for usar o domínio raiz (meusite.com.br), use @ ou consulte a documentação do seu DNS – alguns exigem A/ALIAS em vez de CNAME.
      • 
                
    • Valor/Destino: aponte para o domínio do seu app no Railway, ex: cloaker-pro-novo-production.up.railway.app (veja em Railway → Networking).
      • 
              
    
            
    5. 
            
  5. Aguarde a propagação do DNS (pode levar de alguns minutos a 48 horas). Você pode testar com nslookup cloaker.seudominio.com no terminal.
    6. 
            
  6. Use HTTPS: se seu provedor oferecer SSL (ex: Cloudflare “Flexible” ou “Full”), ative. A URL final deve ser https://cloaker.seudominio.com.
    7. 
            
  7. No painel, em Configurações, preencha o campo acima com a URL completa, por exemplo: https://cloaker.seudominio.com (sem barra no final). Clique em Salvar.
    8. 
            
  8. Pronto. Ao gerar links em Meus Sites ou Link para Ads, os links usarão o prefixo escolhido acima (ex: https://cloaker.seudominio.com/r/XXXXX ou /l/XXXXX).
    9. 
          

          
Cadastre em Domínios, use o destino CNAME no DNS e escolha o domínio ao gerar o link em Meus Sites.
